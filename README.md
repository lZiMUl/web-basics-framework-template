Clone Warehouse Command:
```
git clone https://github.com/lzimul/web-basics-framework-template.git && cd web-basics-framework-template
```

Initialization Command:
```
npm install
```

Development Command:
```
npm run dev
```

Operation Command:
```
npm start
```
